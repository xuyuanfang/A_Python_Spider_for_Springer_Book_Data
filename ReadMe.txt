使用方法
运行“Spider_for_Springer_Book_Data.py”即可
输入相关参数后，请耐心等待处理完成，处理完成时程序窗口会自动关闭
获得的数据结果将以一个名为“Springer_+日期”的表格文件保存在Python程序同一级目录中

Tip：
由于运行一次完整的采集会产生大量的数据请求，花费很多时间，现给出以下参数以供测试之用

Discipline选择1
Language  选择5
start page选择3
start item选择5

如果程序运行正常，你将会得到一个含有4条图书统计数据的Excel文件。
